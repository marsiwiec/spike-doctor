# Changelog

## 0.7.0 (2026-04-27)

Initial (stable) release.

- Batch ABF file processing with whole-cell current-clamp analysis
- Uses the eFEL library for extracting ephys features
- Interactive summary plots (raw traces, F-I curves, phase-plane)
- Export data to CSV, multi-sheet Excel
- Generate multi-page PDFs with graphs for overview
- Docker image and Windows/macOS/Linux binaries via pyinstaller
- Nix flake for reproducible development environment
