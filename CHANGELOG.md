# Changelog

## 0.7.0 (2026-04-27)

Initial stable release.

- Batch ABF file processing with whole-cell current-clamp analysis
- eFEL integration for 200+ electrophysiological features
- Interactive summary plots (raw traces, F-I curves, phase-plane)
- Export to CSV, multi-sheet Excel, and PDF
- Docker image and Windows PyInstaller executable
- Nix flake for reproducible development environment
